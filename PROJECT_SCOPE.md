# MockMate AI — MVP Scope

## Project Goal
MockMate AI helps job seekers practice interviews based on real job vacancies using AI-generated mock interviews and feedback.

---

# Target Users
- First-time job seekers
- Fresh graduates
- Career switchers
- Freelancers

---

# MVP Features
1. Paste job vacancy
2. Generate interview questions
3. Answer questions
4. Receive AI feedback

---

# Interview Mode
Text-based only

---

# Features NOT Included Yet
- Voice interviews
- Video avatars
- Speech emotion analysis
- Multiplayer coaching
- Recruiter dashboard

---

# Core Mission
Reduce interview anxiety and improve confidence through realistic interview practice.
